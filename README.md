# Bajukita Web

Next.js landing/catalog website for Bajukita.my.

This site follows the same frontend approach as the StickerKita website:

- Next.js app router
- Tailwind CSS
- Reusable components
- Static data files
- WhatsApp/email enquiry flow

No database is required for this version.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
